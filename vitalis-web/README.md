# Vitalis Web 🌐

The futuristic frontend for Vitalis OS - built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features

- **Holographic Dashboard**: Glassmorphic cards with real-time AI insights
- **GPU-Accelerated Animations**: Physics-based motion with Framer Motion
- **Bioluminescent Design**: Dark themes with neon accents
- **Responsive Layout**: 12-column grid system

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React 18

## Environment

Create `.env.local`:
```env
NEXT_PUBLIC_API_URL=http://localhost:3000
```
